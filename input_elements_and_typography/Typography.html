Default font (sans-serif): Inter

The default font used by uproot is Inter (v3.19). Inter is a sans-serif font distributed under the SIL Open Font License 1.1. It comes in nine different font weights. In addition to the Latin alphabet, Inter covers the Cyrillic and Greek alphabet.

Monospace font: Cascadia Mono

The default monospace (“typewriter”) font used by uproot is Cascadia Mono. Cascadia Mono is distributed under the SIL Open Font License 1.1. It comes in six different font weights. In addition to the Latin alphabet, Cascadia Mono covers the Arabic, Cyrillic, Greek, and Hebrew alphabet.
