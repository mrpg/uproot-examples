# trust\_game

The trust game (Berg, Dickhaut, & McCabe, 1995). Player 1 (trustor) receives an endowment and can send any amount to Player 2 (trustee). The sent amount is tripled. Player 2 then decides how much to return to Player 1.

Load this app using

```python
load_config(uproot_server, config="trust_game", apps=["trust_game"])
```
