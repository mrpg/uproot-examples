# embed\_static

Load this app using

```python
load_config(uproot_server, config="embed_static", apps=["embed_static"])
```

*Note*: This app ships a [freely licensed](https://web.archive.org/web/20180614105514/https%3A//pixnio.com/fauna-animals/dolphins/animal-dolphin-water-ocean) image depicting two dolphins. See `_static/dolphins.jpg`. The image was released under [CC0](https://creativecommons.org/publicdomain/zero/1.0/deed.en), meaning it can be used in this example without restrictions.
