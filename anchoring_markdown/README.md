# anchoring_markdown

Add the following to your `main.py`:

```python
load_config(uproot_server, config="anchoring_markdown", apps=["anchoring_markdown"])
```

*Note*: This app is exactly identical to `anchoring`, but implemented in Markdown. This is [supported](https://github.com/mrpg/uproot/pull/250) by _uproot_.
