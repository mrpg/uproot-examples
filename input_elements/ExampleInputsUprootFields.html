Example of a “Likert grid”

The likert_grid() macro is the responsive version of the likert_matrix() macro. The rows of radio buttons turn into separate columns of radio buttons if the viewport width falls below a threshold.

The threshold defaults to Bootstrap’s lg breakpoint, that is, 992 pixels. In the following example, the threshold is set to 1200 pixels using breakpoint=1200 for demonstration purposes.

{{ likert_grid(form.likert_field_2, form.likert_field_3, form.likert_field_4, breakpoint=1200) }} {% endblock main %}
