{% extends "Base.html" %} {% block head %}

This is the head block. It is included in the <head> element of the HTML page, that is, before the <body> element.

You must not include visible HTML here but only JavaScript, CSS, and metadata. If this block contains visible text/HTML, it is rendered despite not being part of the <body> element; but doing so violates the HTML standard and can break your experiment in some browsers.

{% endblock head %} {% block pre_container %}

This is the pre_container block. It is included in the <body> element of the HTML page, but before the beginning of the <form> element.

{% endblock pre_container %} {% block wrapper_start %}

This is the wrapper_start block. It is included in the <form> element but before the <div> with class .container.

{% endblock wrapper_start %} {% block start %}

This is the start block. It is included after the start of the <div> with class .container. This is why its width is determined by the settings for the .container class, and it is centered horizontally in the viewport.

{% endblock start %} {% block title %}
This is the title block.

It is embedded in the div with class .container but placed before <main>. Its contents are rended as an <h1> heading.

{% endblock title %} {% block main %}

This is the main block. It is placed within the <main> element.

{% endblock main %} {% block main2 %}

This is the main2 block. It is placed within the <main> element.

{% endblock main2 %} {% block main3 %}

This is the main3 block. It is placed within the <main> element.

{% endblock main3 %} {% block postmain %}

This is the postmain block. It is placed after the <main> element.

{% endblock postmain %} {% block wrapper_end %}

This is the wrapper_end block. It is placed after the <main> element but still within the <form> element.

As illustrated here, you can use a block like this to produce a full-width element, such as a block with a background spanning the entire width of the browser window. This is a very common approach in current web design. If you still want the cotent of this block to have the width of the main content and be horizontally centered like the main content—as this paragraph is—just apply the .container class to the content.

{% endblock wrapper_end %} {% block late %}

This is the late block. It is placed after the <form> element.

{% endblock late %} {% block late2 %}

This is the late2 block. It is placed after the <form> element.

{% endblock late2 %} {% block footer %}

This is the footer block. It is included in the <footer> element. By default, it contains an empty <div> with a specified height to prevent buttons from being placed too close to the bottom of the browser window (which can cause them to be unclickable, in particular, on smartphones).

{% endblock footer %}
