{% extends "Base.html" %} {% block head %}

This is the head block.

It is included in the <head> element of the HTML page, that is, before the <body> element.

You should not include HTML here but only JavaScript, CSS, and metadata. If this block contains text/HTML, it is rendered despite not being part of the <body> element, as you can see here.

{% endblock head %} {% block pre_form %}

This is the pre_form block.

It is included in the <body> element of the HTML page, but before the beginning of the <form> element.

{% endblock pre_form %} {% block header_start %}

This is the header_start block.

It is placed at the start of the <header> element, that is, before uproot’s timeout indicator. The header element is placed before the form element.

By default, it is as wide as the body of your document, which is the full width of the browser window by default.

If you want some content of this block to have the width of the main content and be horizontally centered like the main content—as this paragraph is—just apply the .container class to that content.

{% endblock header_start %} {% block header_end %}

This is the header_end block.

It is placed at the end of the <header> element, that is, after uproot’s timeout indicator.

{% endblock header_end %} {% block form_start %}

This is the form_start block. It is included at the start of the <form> element.

{% endblock form_start %} {% block pre_main %}

This is the pre_main block.

It is placed before the <main> and the <h1> heading and embedded in a <div> with class .container.

If you want some content of this block to have the width of the main content and be horizontally centered like the main content—as this paragraph is—just apply the .container class to that content.

{% endblock pre_main %} {% block title %}
This is the title block.

It is embedded in a div with class .container and placed before <main>. Its contents are embedded in an <h1> heading.

{% endblock title %} {% block main %}

This is the main block.

It is placed within the <main> element. It is embedded in a <div> with class .container. Hence, its width is determined by the settings for the .container class, and it is centered horizontally in the viewport.

{% endblock main %} {% block main_full_width %}

This is the main_full_width block.

It is placed within the <main> element. It has the width of the <main> element, which is the width of the <body> element by default. This, in turn, is the full width of the browser window by default. You can, thus, use this block, for instance, to highlight content with a color band. This is a very common approach in current web design.

If you want some content of this block to have the width of the main content and be horizontally centered like the main content—as this paragraph is—just apply the .container class to that content.

{% endblock main_full_width %} {% block main2 %}

This is the main2 block.

It is placed within the <main> element and embedded in a <div> with class .container.

{% endblock main2 %} {% block main2_full_width %}

This is the main2_full_width block.

It is placed within the <main> element and has full width.

{% endblock main2_full_width %} {% block main3 %}

This is the main3 block.

It is placed within the <main> element and embedded in a <div> with class .container.

{% endblock main3 %} {% block post_main %}

This is the post_main block.

It is placed after the <main> element, still within the <form> element.

{% endblock post_main %} {% block form_end %}

This is the form_end block.

It is placed after the <main> element, right before the end of the <form> element.

{% endblock form_end %} {% block late %}

This is the late block.

It is placed after the <form> element and mainly intended for including JavaScript.

{% endblock late %} {% block late2 %}

This is the late2 block.

It is placed after the <form> element and mainly intended for including JavaScript.

{% endblock late2 %} {% block footer %}

This is the footer block.

It is included in the <footer> element. By default, it contains an empty <div> with a specified height to prevent buttons from being placed too close to the bottom of the browser window (which can cause them to be unclickable, in particular, on smartphones).

If you want some content of this block to have the width of the main content and be horizontally centered like the main content—as this paragraph is—just apply the .container class to that content.

{% endblock footer %}
